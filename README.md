Women Health Tracker Project
Women Health Tracker is a Java-based project designed as a simple application to track and learn about various women’s health problems like PCOS, Endometriosis, Hormonal Acne, and Hair Fall problems. The main idea behind developing this project is to integrate basic health knowledge with programming in a simple way.
The system accepts input from users regarding their health problems and then provides a basic analysis along with some suggestions. It is not designed to diagnose any problem; rather, it is designed to provide a basic idea and spread awareness about these problems.
Features
The project is designed as a simple menu-driven system in which users can input their data and obtain useful output from the system. It tracks various things like cycle length, pain level, acne level, hair fall problems, irregular cycles, and fatigue.
The system provides possible indications of problems like PCOS and Endometriosis based on the input provided by users. It also provides suggestions like maintaining a healthy lifestyle and consulting a doctor in case of problems. There is a facility in the system to spread awareness about these problems by providing information regarding these problems.
The code is segmented into different packages. This makes it easier to understand the code.
How the system works
The user is first required to fill in some basic information such as age, among other health-related information. The information is then analyzed through simple conditions in the program.
The program then analyzes the information by checking for patterns such as long cycle length, high pain levels, among other symptoms such as acne and hair fall. It then provides a simple analysis of the information.
It also has the ability to save information for different users, which can then be viewed later.
Technologies used
The program is written in Java. It mainly incorporates simple programming concepts.
It incorporates simple concepts of object-oriented programming. It has features such as class, object, among others. It also incorporates the use of ArrayList for storing user information. It also incorporates simple condition-based programming.
Project structure
The program is segmented into different packages.
The model package stores user information and health data.
The services package contains the logic for performing user analysis.
The data package contains methods to store and retrieve user information.
The utils package contains various validation methods.
The awareness package provides information on various conditions.
The ui package contains the main class to interact with users.
Steps to run the project
Ensure you have Java installed on your system.
Open the project in IntelliJ or any other integrated development environment.
Execute the MainApp.java class.
Follow the menu-driven options in the console.
You can store user information, retrieve data, and perform health analysis.
Example
If a user inputs pain levels along with irregular cycles, the system will suggest possible signs of various conditions such as PCOS or endometriosis, recommending them to consult a doctor or improve their lifestyle habits.
Upcoming enhancements
This project can be enhanced in the future by utilizing JavaFX or Swing to create a graphical user interface. A database can also be used to store data in place of memory. Features like tracking history or utilizing external APIs can be added, and the analysis can be improved in accuracy.
